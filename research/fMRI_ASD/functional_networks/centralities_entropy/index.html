Betweenness Centrality¶

Betweenness centrality of a node $v$ is defined as fraction of sum of all-pair shortest paths that pass through $v$. \begin{equation*} c_B(v) =\sum_{s,t \in V} \frac{\sigma(s, t|v)}{\sigma(s, t)} \end{equation*} where $V$ is the set of nodes, $\sigma(s, t)$ is the number of shortest $(s, t)$-paths, and $\sigma(s, t|v)$ is the number of those paths passing through some node $v$ other than $s, t$. If $s = t$, $\sigma(s, t) = 1$, and if $v \in {s, t}, \sigma(s, t|v) = 0$

Closeness Centrality¶

For a connected graph, closeness centrality of a node $v$ is defined as inverse of sum of the shortest distance between $v$ and other $n-1$ nodes. \begin{equation*} C(u) = \frac{1}{\sum_{v=1}^{n-1} d(v, u)} \end{equation*} where $d(v, u)$ is the length of shortest path between node $v$ and $u$.

In other variation, above equation is scaled of by a factor of $n-1$, so as to represent average shortest distance of given node in the graph.

However, for disconnected graphs, there are various variants, like, taking sum of inverse of the shortest distance instead of inverse of the sum. This allows us to make $d(v, u) = \infty$, if $v$ and $u$ are not connected.

Eigenvector Centrality¶

Eigenvector centrality is used as a ranking measure. It helps to find which nodes are important. Let $A$ be the adjacency matrix of graph $G$, $\lambda$ be the largest eigen-value of $A$ in absolute value. Let $x$ be the corresponding eigenvector. Then $EEC(i)$ for node $i = x_i$.

Network Spectral Entropy¶

Network entropy is a measure of randomness of the graph. For this we need to generate various Erdos-Renyl graph, as the entropy measurement requires an expectation value.

In [21]:
n = 10
p = 0.5
nsamples = 10000
vs = np.array([0]*n)
for i in range(0,nsamples):
    G = nx.erdos_renyi_graph(n,p)
    A = nx.to_numpy_matrix(G)
    v = np.real(eigvals(A))
    vs = vs + v
vs = vs/nsamples
In [22]:
vs = vs/(np.sqrt(n))
kde = stats.gaussian_kde(vs,bw_method='silverman')
x = np.linspace(vs.min(), vs.max(), 100)
rho = kde(x)
